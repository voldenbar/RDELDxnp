# 项目简介

## 核心功能
- 用户注册与登录
- 首页导航：系统首页、宠物医院、药品信息、医生信息、公告信息、个人中心
- 个人中心：更新个人详细信息、修改密码、宠物信息管理、预约医生、医生诊断、提交药品、取消药品、用药反馈、反馈处理、我的收藏
- 管理员登录：主页、用户管理、宠物医生管理、宠物品种管理、宠物信息管理、宠物医院管理、药品分类管理、药品信息管理、医生信息管理、预约医生管理、医生诊断管理、提交药品管理、取消药品管理、用药反馈管理、反馈处理管理、系统管理、个人中心
- 宠物医生后台管理：主页、用户管理、宠物信息管理、宠物医院管理、药品信息管理、预约医生管理、医生诊断管理、提交药品管理、取消药品管理、用药反馈管理、反馈处理管理、个人中心

## 适用场景
- 宠物医院内部药品管理
- 宠物主人预约医生和提交药品信息
- 医生诊断和用药反馈处理
- 系统管理员进行医院和药品信息维护

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
